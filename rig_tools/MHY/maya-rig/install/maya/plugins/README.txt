How to install maya plug-ins:  (* This is a temp method)

1) Open the correct Maya version folder, for example: "MAYA2022"
2) Download plugins in the folder:
	cmt.mll
	mhy.mll
2) Got to your local maya folder, then create a new folder named "plugins":
	C:\Users\{name}\Documents\maya\{version}\plugins
3) Move downloaded mll files into the plugins folder.
4) Open same version of maya, in top menu, go to Windows/Settings/Preference/Plug-in Manager.
5) In Plug-in Manager, click "Browse" to load the .mll files.
6) Turn on both "Loaded" and "Auto load", then close the UI.
